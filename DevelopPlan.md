# AI旅行规划师 - 两周开发工作计划

## 📋 项目概述

**开发时间**：14天  
**技术栈**：Vue 3 + Python FastAPI  
**目标**：实现MVP版本，重点完成核心功能  
**开发者**：编程新手友好版本

## 🎯 MVP功能范围定义

### ✅ 核心功能（必须实现）
1. **基础用户系统**
   - 简单的用户注册/登录
   - 用户信息管理

2. **智能行程规划**
   - **语音输入旅行需求**（必须实现）
   - 文字输入旅行需求（备选方案）
   - AI生成个性化行程方案
   - 行程展示和简单编辑

3. **基础预算管理**
   - 预算设定
   - **语音记录费用**（推荐实现）
   - 手动费用记录
   - 预算使用情况展示

4. **基础地图展示**
   - 集成地图API
   - 显示行程地点
   - 基础位置信息

5. **云端数据同步**
   - 用户数据云端存储
   - 多份旅行计划管理

### 🔄 高级功能（时间允许时实现）
- 复杂的行程优化算法
- 详细的费用分析图表
- 社交分享功能
- 离线数据缓存

## 🛠️ 技术栈详细配置

### 前端技术栈
- **框架**：Vue 3 + Composition API
- **构建工具**：Vite
- **UI组件库**：Element Plus
- **状态管理**：Pinia
- **路由**：Vue Router 4
- **HTTP客户端**：Axios
- **地图**：高德地图 Web API
- **语音识别**：科大讯飞 Web API（核心功能）
- **AI集成**：阿里云百炼API

### 后端技术栈
- **框架**：Python FastAPI
- **数据库**：SQLite（开发阶段）
- **ORM**：SQLAlchemy
- **认证**：JWT
- **API文档**：FastAPI自动生成
- **CORS**：FastAPI CORS中间件

### 开发工具
- **代码编辑器**：VS Code
- **版本控制**：Git + GitHub
- **API测试**：Postman 或 FastAPI自带文档
- **包管理**：npm (前端) + pip (后端)

## 🔧 环境配置详细步骤

### 1. 基础软件安装

#### 1.1 安装 Node.js
```bash
# 访问 https://nodejs.org/
# 下载并安装 LTS 版本（推荐 v18+）
# 验证安装
node --version
npm --version
```

#### 1.2 安装 Python
```bash
# 访问 https://www.python.org/
# 下载并安装 Python 3.9+
# 验证安装
python --version
pip --version
```

#### 1.3 安装 VS Code
```bash
# 访问 https://code.visualstudio.com/
# 下载并安装
# 推荐插件：
# - Vue Language Features (Volar)
# - Python
# - GitLens
# - Thunder Client (API测试)
```

#### 1.4 安装 Git
```bash
# 访问 https://git-scm.com/
# 下载并安装
# 配置用户信息
git config --global user.name "你的姓名"
git config --global user.email "你的邮箱"
```

### 2. 项目初始化

#### 2.1 创建项目目录
```bash
mkdir AI-Travel-Planner
cd AI-Travel-Planner
```

#### 2.2 初始化前端项目
```bash
# 创建 Vue 项目
npm create vue@latest frontend
cd frontend
npm install

# 安装额外依赖
npm install element-plus axios pinia @element-plus/icons-vue
npm install @amap/amap-jsapi-loader  # 高德地图
npm install js-audio-recorder  # 语音录制
```

#### 2.3 初始化后端项目
```bash
# 回到根目录
cd ..
mkdir backend
cd backend

# 创建虚拟环境
python -m venv venv
# Windows 激活虚拟环境
venv\Scripts\activate
# 安装依赖
pip install fastapi uvicorn sqlalchemy sqlite3 python-jose[cryptography] passlib[bcrypt] python-multipart
```
