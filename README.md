# AI-Travel-Planner
您的 AI 旅行规划师

## 技术栈
- **前端**: Vue 3 + TypeScript + Vite + Element Plus
- **后端**: FastAPI + SQLAlchemy + SQLite
- **开发工具**: Docker, Git

## 快速开始

### 后端启动
```bash
cd backend
pip install -r requirements.txt
python start.py
```
访问: http://localhost:8000/docs

### 前端启动
```bash
cd frontend
npm install
npm run dev
```
访问: http://localhost:5173

## 项目结构
```
├── backend/          # FastAPI 后端
├── frontend/         # Vue 3 前端
├── docker-compose.yml
└── README.md
```
